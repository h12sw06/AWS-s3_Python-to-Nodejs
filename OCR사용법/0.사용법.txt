모든 내용은 영상으로 제작되어 있습니다. '넣은 경로 위치\OCR\OCR사용법\영상' 확인 바랍니다.
(영상에 있는 사용법은 옛날 사용법입니다. 무시하셔도 됩니다.)

1. '바탕화면 -> OCR -> 0.사용법 - 바로 가기' 파일열기
2. '1.자동설치 - 바로 가기' 실행
(아무키나 누르기전에 파일 이름만 나올시 다시 '1.자동설치 - 바로 가기' 다시실행 [전체경로 필요!!])
3. 참조된 영상 확인 (위치 : OCR -> OCR사용법 -> 영상)
4. 설치 진행(각각 설치 방법 참고자료 나옴)
 「참고 : 1. 설치 되어 있는 거는 중복해서 설치할 필요 없음(제어판에서 확인 바람)
           2. Node.js 설치할때 "Automatically install the necessary tools...." 라고 하는 문구는 절대 체크해제
           3. Tesseract_OCR설치후 환경변수 설정 안해도됨(선택사항)
           4. 나머지는 또같이 따라하기(또같이 안할시 안될수도 있음)」
